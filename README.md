# Prof. Jang-Ping Sheu Homepage

This repository contains the source code for Prof. Jang-Ping Sheu's homepage, refactored into a modern, static HTML structure.

## Overview

The original website (http://hscc.cs.nthu.edu.tw/~sheujp/) was refactored to use semantic HTML5 and centralized CSS.

## File Structure

- `index.html`: The main landing page.
- `english.html`: Curriculum Vitae (English).
- `chinese.html`: Curriculum Vitae (Chinese).
- `publication.html`: Comprehensive list of publications (Journal, Conference, Books).
- `research.html`: Overview of research areas.
- `student_project.html`: Student projects and videos.
- `award_list.html`: Awards received by students.
- `album.html`: Photo album.
- `css/style.css`: Main stylesheet for the website.
- `images/`: Directory containing all image assets used on the site.

## Usage

Simply open `index.html` in any modern web browser to view the site.

## Changes from Original

- Converted table-based layout to CSS-based layout.
- Semantic HTML tags (`<header>`, `<nav>`, `<main>`, `<footer>`) are used for better accessibility and SEO.
- External links (e.g., to PDF files or other websites) are preserved.
